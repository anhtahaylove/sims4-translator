## Recommended Public Release Path

1. Open package or STBL files.
2. Translate and review strings.
3. Fix token warnings in Translation Studio.
4. Run Validate Release.
5. Use Save as package.
6. Test the package in the Mods folder before sharing.

This project is a community localization tool. It does not include official EA, Maxis, or The Sims artwork, logos, fonts, or game assets.

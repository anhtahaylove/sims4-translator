## Project

**The Sims 4 Translator Plus** is a community-maintained fork of:

- Upstream project: `voky1/sims4-translator`
- Upstream URL: https://github.com/voky1/sims4-translator
- Fork URL: https://github.com/anhtahaylove/sims4-translator

translation file compatibility goals. The fork adds stabilization work, async
